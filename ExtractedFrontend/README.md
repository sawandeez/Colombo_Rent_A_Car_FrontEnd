# Extracted Frontend

Extracted frontend for car rental system featuring vehicle types and vehicle listing.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- ✅ Vehicle Types API Integration
- ✅ Vehicle Listing with Type Filter
- ✅ Responsive Design with Tailwind CSS
- ✅ React Query for Data Fetching
- ✅ Framer Motion Animations

## Environment

Create `.env` file:

```
VITE_API_BASE_URL=http://localhost:8080/api
```
